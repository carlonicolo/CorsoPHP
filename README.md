# PHP Tutorial Examples

This repository contains basic PHP examples covering fundamental programming concepts:

## Contents

- **[Variables](variables.php)** - Variable declaration and usage in PHP
- **[If...else](ifelse.php)** - Conditional statements and branching logic
- **[Switch](switch.php)** - Switch-case control structure
- **[While](while.php)** - While loop implementation
- **[Do...while](dowhile.php)** - Do-while loop structure
- **[For](for.php)** - For loop examples
- **[Array](array.php)** - Array creation and manipulation
- **[Funzioni con e senza parametri](function.php)** - Functions in PHP: with and without parameters, return values, and scope
- **[Form - GET e POST](form.php)** - Form handling with GET and POST methods, form validation, and data processing
- **[Read, write, scandir, upload files](filereadwrite.php)** - File system operations: reading, writing, directory scanning, and file upload functionality

## Usage

Each file demonstrates a specific PHP concept with practical examples. Simply run the PHP files on a server with PHP support to see the output.

## Requirements

- PHP 7.0 or higher
- Web server (Apache, Nginx, or built-in PHP server)
